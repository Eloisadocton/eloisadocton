SKETCHES

I typically create a sketch any time I'm developing, this helps me think about the user experience, discuss it with my team mates, tinker, and develop. It's an easy and quick way to see if something will work or not.

WIREFRAMES

Once I'm done with sketching, I clean out the work by putting together some low or high fidelity wireframes. This really helps iron out details of the user journey.
